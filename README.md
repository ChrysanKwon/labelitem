# LabelItem

A simple YOLO image annotation tool built with PySide6.

## Origin
Originally a user of labelImg, I wanted to create a personalized version with a refined workflow. This project was developed as a hands-on experiment using Claude Code to build a custom labeling solution from scratch.

## Quick Start

### Installation
pip install PySide6

### Usage
python main.py

## Hotkeys
- Navigation: A / D or Arrow keys
- Delete Box: Delete / Backspace
- Confirm Class: Enter

## Project Structure
- main.py: Core logic
- ui_layout.py: UI design
- io_labels.py: YOLO I/O
- config.py: Session and settings
